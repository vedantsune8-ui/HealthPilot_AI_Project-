# HealthPilot_AI_Project-
HealthPilot AI is an AI-powered healthcare web application built using Python, Flask, Machine Learning, and SQLite. It helps users check symptoms, predict possible diseases, calculate BMI, set medicine reminders, and generate health reports. It also includes an admin dashboard for managing users and health data.
HealthPilot AI – Short Project Detail

HealthPilot AI is an AI-based healthcare web application that helps users monitor their health and get basic medical insights. It can check symptoms, predict possible diseases, calculate BMI, generate health reports, and set medicine reminders. It also has an admin dashboard for managing users and health data.

Languages / Technologies Used:
Python → Main backend logic
Flask → Web framework for building the application
HTML → Structure of web pages
CSS → Styling and UI design
JavaScript → Interactive features
SQLite → Database to store user and health records
Machine Learning (Python libraries like Scikit-learn, Pandas, NumPy) → Disease prediction model
Main Features:

✔ Symptom Checker
✔ Disease Prediction
✔ BMI Calculator
✔ Medicine Reminder
✔ Health Report
✔ Admin Dashboard

This project is useful for AI + Healthcare + Web Development learning and looks strong for a mini/final-year project
